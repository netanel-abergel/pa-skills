# Brand Styling — monday Builders

Sourced from the 2026 Builders Kickoff master deck (402 pages, 16:9, dark).

## Colors
| Token | Hex | Usage |
|---|---|---|
| `--y` | `#FBC926` | Yellow — primary brand accent |
| `--g` | `#59E36C` | Green — acceleration, "we are it" |
| `--c` | `#4DDDD5` | Cyan |
| `--b` | `#5C5CE0` | Indigo — filled key boxes |
| `--p` | `#9B7BFC` | Purple |
| `--pink` | `#FF158A` | Hot pink |
| `--red` | `#DF2F4A` | Incidents / critical |
| `--bg` | `#000000` | Dark (default) |
| `--card` | `#1A1A1A` | Card fill on dark bg |

## Signature Gradient
```css
linear-gradient(180deg, #FBC926 0%, #59E36C 33%, #4DDDD5 66%, #9B7BFC 100%)
```
Used for: card outlines, cover ribbons, page number circles, dividers. Always vertical unless following a curved path.

## Typography
Family: `'Figtree', 'Inter', system-ui, sans-serif`
- Marquee headlines: weight 200, 60-100pt, left-aligned
- Slide titles: weight 300-400, 36-48pt
- Card headings: weight 600, 20-28pt
- Body: weight 400-500, 14-18pt
- Eyebrows: weight 500, uppercase, 12-14pt, yellow, letter-spacing 0.15em
- Stat numerals: weight 700, 80-160pt

## Compositional Rules
- **Lean left** — headlines anchor left, right 30-40% is breathing space
- **One color per slide** — pick ONE accent and use consistently
- **Gradient is the brand** — reserve for hero moments only
- **Negative space > density** — drop content rather than shrink
- **Cover ribbons on cover ONLY**
- **Logo on cover and dividers ONLY**
- **Dark theme default** — light is for external shareholder PDFs only
